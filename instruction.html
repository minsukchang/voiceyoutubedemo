Instruction 1/8 - Reading all the Instructions

In this experiment, we want to see how using voice to navigate how-to videos can help people accomplish their tasks.

There are total EIGHT instructions, Please read them all before diving into the tasks

Instruction 2/8 - Getting the Tools Ready

YOU HAVE TO WATCH, DO AND LEARN SIMULTANEOUSLY.

So please have the necessary tools ready at your hands, so you can follow the video while doing the tasks.

Do you have the tools with you now?

Instruction 3/8 - Allowing Microphone Access

When you visit https://www.minsukchang.com/voiceyoutubedemo/, the website will prompt about granting microphone access.

Please click “ALLOW”. You will need this in order for the interface to listen to you.

Instruction 4/8 - Getting My Video in the Interface

once you have found a youtub video you want to follow, copy the URL of your YouTube video from the address bar at the top

paste in the URL of the video you found in the interface and click (or say) “change video”

If the video changes and starts playing, say “stop”

If these steps worked, your microphone is working with the interface. If it does not work, please refresh the page, and follow the steps 3 and 4 again.

Instruction 5/8 - Interface Functions

On the interface, you can control the video using voice, play, pause, stop, go back or skip by a few seconds or minutes, and more.

At any point if you think the interface is not listening to you, you can click on the “start listening again” button.

If the interface does not support what you want to do, say it how you want to be able to say. And carry out the task using mouse, keyboard, or voice.

Example.

  • my need: "I want to see the beginning of the step, because I want to check if I have it right"
  • I want to be able to say: "I want to see that again"
  • How I did it: used mouse to scrub the timeline and found the beginning of step

Instruction 6/8 - Wearing a Headset

Please wear a headset to watch the video and follow the tasks

Instruction 7/8 - Picking a Different Video

Feel free to pick another video, if first video is not good for you at any point in the study.

Just find another video from YouTube and paste the URL.
