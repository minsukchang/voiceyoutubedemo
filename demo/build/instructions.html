Introduction

In this demo, we want to see how people use voice to navigate HOW-TO VIDEOS.

On the interface, you can control the video using voice. We are deliberately not telling you the exact commands, because we want to see how YOU navigate the video and how YOU say them

Click on the “Start Talking button and you will see the transcript of what you say.

THIS IS HYPOTHETICAL, SO DON'T WORRY ABOUT SAYING IT IN A WAY A SYSTEM WOULD UNDERSTAND. Just say HOW YOU want to say it.

Example

  • my need: "I want to see the part where she adds salt to check how much I need it"
  • I want to be able to say: "Where's the salt?"
  • my need: "I want to see move a bit forward"
  • I want to be able to say: "30 seconds before"

Instruction 1/6 - Use Chrome Browser

YOU HAVE TO USE CHROME TO TEST THE INTERFACE.

Our system uses "react-speech-recognition", a React component that converts speech from the microphone to text.

Under the hood, it uses Web Speech API. Currently, this component will only work in Chrome. It fails gracefully on other browsers.

Instruction 2/6 - Allowing Microphone Access

When you visit our interface (Don't do it now), the website will prompt about granting microphone access.

Please click “ALLOW”. You will need this in order for the interface to listen to you.

Instruction 3/6 - Changing Video in the Interface

You can change the video to watch using the MENU ON TOP RIGHT

If you want to request videos not on the list, please contact minarainbow@kaist.ac.kr

Instruction 4/6 - Wearing a Headset

Please wear a headset to watch the video and follow the tasks.

Click START LISTENING to start controlling the video using voice.

Instruction 5/6 - Using history and bookmarks

After each jump, you can say RETURN to go to where you were before last jump.

Also, "some" of your navigation records will be added in below table. You can revisit this point by saying NUMBER (ID #).

While watching the video, you can also manually add bookmarks if you feel like it is likely to revist the frame.

Say ADD BOOKMARK and GO TO NEXT/PREVIOUS BOOKMARK

Instruction 6/6 - Evaluating the navigation

After you navigate the video, an EVALUATION POPUP will show up asking whether the system has jumped correctly.

Please take at least few seconds of break between each commands. Also, simple commands might not prompt this evaluation task.

Unless the resulting frame is the exact frame you expected, please let us know the details about the failure.

This cannot be done by voice, you have to use mouse and keyboard.
